⚠️ POLICY CHANGE

Effective February 1, 2026

SSA Processing Method Change

Click to view official documentation: Supplier SSA Allowance Change Notification

Key Policy Change

SSA Will No Longer Be Taken Off-Invoice

NEW PROCESS (Effective Feb 1, 2026):

  • UNFI will deduct SSA monthly through accounts payable
  • One deduction per region (East/West)
  • Based on prior-month PO receipts
  • Calculation: PO receipts × SSA rate of 2.5%
  • UNFI will send backup when deduction hits

What Stays the Same

These allowances continue as off-invoice:

  • Promo allowances (where applicable) - remains off-invoice
  • Freight allowances - remains off-invoice
  • Spoils allowances - remains off-invoice

No changes to how these are processed today

UNFI's Rationale for Change

  • Cleaner deductions - simplified processing
  • Fewer SSA lines on checks - reduced clutter
  • Easier reconciliation - one monthly deduction per region
  • Simplified backup documentation - streamlined tracking

Required Actions - Compliance

Swoffle must complete the following by February 1, 2026:

1. Stop Giving SSA Off-Invoice

Effective February 1, 2026 - no more SSA deductions on invoices

2. Remove SSA EDI Codes

  • Remove code H000 from 810s (invoices)
  • Remove code 50 from 880s (grocery products invoice)

3. Error Correction Protocol

If SSA is accidentally given off-invoice after Feb 1, email UNFI's dispute team immediately

4. Confirm with SRM

Notify Swoffle's Senior Relationship Manager (SRM) that changes have been implemented
